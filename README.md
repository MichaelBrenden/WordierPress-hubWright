# WordierPress-hubWright

WordierPress is a community of communities that maximally benefits wordsmiths and lyriphiles, content and humanity.

Installed on participants' existing wordpress sites, WordPress hubWright connects, shares, grows.

All code is 100% open-source, written and backed by wordierpress.com

You can pay us to host your goods, or you can host any part (or all parts) yourself, optionally including running your own hub/publishring.

The goal is No More Dependency upon or trust for massively centralized (and thus controllable and controlling) platforms such as Substack, Patreon, Facebook, etc.
